# Counterfeit Medicine Blockchain

A blockchain-based solution for tracking and verifying pharmaceutical products to combat counterfeit medicines.

## Features

- Medicine verification system
- Blockchain-based tracking
- Dashboard for stakeholders
- DDA (Drug Distribution Authority) view

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase
- React Router

## Getting Started

1. Clone the repository
2. Install dependencies: `bun install`
3. Copy `.env.example` to `.env` and fill in your Supabase credentials
4. Run development server: `bun run dev`

## Project Structure

- `/public` - Static assets
- `/src/components` - Reusable UI components
- `/src/pages` - Route-level page components
- `/src/hooks` - Custom React hooks
- `/src/lib` - Utility functions
- `/src/integrations` - External service integrations
- `/supabase` - Backend configuration
